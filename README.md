# 商超库存管理系统 💼

> 基于 SpringBoot + HTML + Chart.js 的混合架构系统

## 🧩 功能特性
- ✅ 商品管理：增删改查
- ✅ 入库 / 出库 操作
- ✅ 库存预警：自动提醒低库存商品
- ✅ 图表分析：分类统计 + 趋势图
- ✅ Excel 导入导出
- ✅ 操作日志记录与追溯

## 📦 技术栈
| 类型   | 技术                    |
| ------ | ----------------------- |
| 后端   | Java + SpringBoot       |
| 前端   | HTML + CSS + JavaScript |
| 可视化 | Chart.js                |
| 构建   | Maven                   |
| 部署   | Live Server / Electron  |

## 🚀 使用方法
1. 启动后端：`mvn spring-boot:run`
2. VS Code 打开 `/web` → Open with Live Server
3. 浏览器访问：`http://127.0.0.1:5500`
